# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in LightWeightChartsMCP, please report it responsibly:

1. **Do NOT open a public GitHub issue.**
2. Email: contact@fractalyst.dev with a description and reproduction steps.
3. You will receive a response within 48 hours.

## Scope

- Vulnerabilities in the MCP server, tools, or validator that could leak data or execute arbitrary code.
- Vulnerabilities in the data pipeline that could corrupt the vector store.

## Out of Scope

- Vulnerabilities in upstream dependencies (report to the respective maintainers).
- Issues requiring physical access to a user's machine.

## Disclosure

Once a fix is released, we will publish a security advisory on GitHub.
